Logo Usage Rules

Every time the logo appears on a B Clean surface, these rules apply.

✔ Do

Use the dark variant on light backgrounds

Warm white (#FCFAF6), pure white, or cream (#F4EFE3). The original teal & graphite logo is designed for these surfaces.

✔ Do

Use the light variant on dark backgrounds

Charcoal (#1C2329) or slate surfaces. Cream replaces graphite; the teal stays the same — it passes AA on dark.

✔ Do

Respect the clearspace

Minimum 1x the height of the wordmark on all sides. Don't crowd with other elements.

✘ Don't

Recolor the logo

The teal (#00A8B0) is brand equity. Never swap it for another color even as an experiment.

✘ Don't

Rotate, skew, or stretch

Always display at its native proportions. No distortion, no tilted angles.

✘ Don't

Place on busy photography

Always use a solid background or a strong flat color overlay. Never float over a busy before/after photo.

Color Usage Rules

One teal, deeper and more saturated than the original. Burnt teal works as a background color with cream or white text, and passes WCAG AA at every text size.

— Deep Teal Rule
#0A4F5C is the one teal in the system. Use it for backgrounds, buttons, headlines, links, icons, and ornament — every text size passes AA on cream (4.94:1) or white (5.35:1).
— B Clean Teal Rule
#00A8B0 is for 24px+ text on cream and body text on charcoal. Never body text on cream. Use for tags, large labels, and secondary accents.
— Graphite Rule
#F4EFE3 is decorative only at any size. Use for dividers, ornaments, and illustrated accents. Never text. Never interactive.

About B Clean Carpet & Upholstery

B Clean Carpet & Upholstery is a Central Ohio custom carpentry studio built on one belief: the work should outlive the room it's in. For two decades we've specialized in high-end residential remodeling — custom kitchens, finished basements, and built-in millwork for homeowners who care about how a space is built, not just how it looks on opening day.

We take the time the work requires. Every joint is tuned, every reveal measured twice, every finish applied with the understanding that a homeowner is going to run their hand across it for the next twenty years.

"Excellent job. We are very satisfied. Easy, fast and friendly at a great price. Highly recommend!"

— Our working philosophy

What we build

We focus our craft on three categories of residential work so we can go deep on each one instead of spreading thin across the whole trade.

  • Custom Kitchens — hand-built cabinet boxes, tuned reveals, integrated hardware, heirloom-grade finishing
  • Finished Basements — warm, dry, properly insulated living space that feels like the rest of the house
  • Built-ins & Millwork — library walls, window seats, mantels, and trim carpentry
  • Licensed & insured in the State of Ohio with a 5-year workmanship warranty
  • Every project includes a detailed written scope before a single tool comes out of the van

Our process

Good carpentry is slow carpentry done with intention. That means fewer projects per year, more care per joint, and a build cadence that keeps you informed at every step.

  1. On-site consultation — we walk the space, ask how you live in it, and listen more than we talk
  2. Written scope & design — exact materials, finishes, dimensions, and a fixed timeline
  3. Material selection — samples in the room you're building, not a showroom
  4. Build & finish — in-house carpenters, never subbed out
  5. Walk-through & warranty — you don't sign off until every joint is one you'd show another carpenter

Ready to talk about a project? Book a free on-site consultation and we'll walk the room with you.

CSS Variables

/* ===== B Clean Carpet & Upholstery — Clean Modern Trust & Dark Drama Tokens ===== */
:root {
  /* Colors */
  --bg:             #FCFAF6;   /* cream — primary canvas */
  --bg-alt:         #FFFFFF;   /* white — card surfaces on cream */
  --bg-dark:        #1C2329;   /* dark charcoal — dark sections, footer */
  --text:           #2A3540;   /* charcoal — primary body copy */
  --text-sec:       rgba(42,53,64,0.72);   /* charcoal-muted — captions, meta */
  --text-muted:     rgba(42,53,64,0.58);   /* lighter charcoal-muted */
  --accent:         #0A4F5C;   /* deep teal — large text, buttons, ornament */
  --accent-dark:    #0A4F5C;   /* deep teal — small text, inline links (AA safe) */
  --border:         rgba(42,53,64,0.14);   /* warm cream border */
  --border-light:   rgba(42,53,64,0.08);   /* lighter warm border */
  --green:          #00A8B0;   /* sage — nature accent, large text highlights */
  --sage:           #F4EFE3;   /* warm cream — decorative only, never text */
  --charcoal:       #1C2329;   /* charcoal — logo ink */
  --white:          #ffffff;   /* pure white — cards, overlays */

  /* Typography */
  --font-display: 'Outfit', Georgia, serif;                         /* headings, accents */
  --font-accent:  'Outfit', Georgia, serif;                         /* italic eyebrows, quotes */
  --font-body:    'Inter', system-ui, -apple-system, sans-serif;      /* body, nav, UI */

  /* Type Scale */
  --text-hero:    clamp(44px, 6vw, 76px);  /* H1 hero display — Outfit 500 */
  --text-section: 2.4rem;     /* 38px — section titles */
  --text-card:    1.5rem;     /* 24px — card headings */
  --text-body:    1.125rem;   /* 18px — body copy */
  --text-small:   0.875rem;   /* 14px — meta, captions */
  --text-label:   0.75rem;    /* 12px — labels, caps */

  /* Spacing */
  --space-section:   80px;   /* between major sections */
  --space-card:      32px;   /* inside cards and panels */
  --space-grid:      24px;   /* between grid items */
  --space-component: 16px;   /* inline component spacing */

  --radius-md:   10px;
